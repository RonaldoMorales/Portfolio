Proyecto Destacado

Ayuda a la Gestión de Recursos Mineros

Este proyecto tuvo como objetivo ayudar a una empresa minera a gestionar sus recursos, se tuvo que aplicar estructuras de datos para resolver esta problemática, la realización de este taller es uno de los logros más importantes de mi carrera universitaria, debido a la dificultad de el problema planteado.

Rol: Desarrollador Principal
Logros: Implementación de un árbol binario y un max heap para gestionar los recursos, mejorando la eficiencia en la asignación de tareas y reducción de tiempos operativos.

Tecnologías utilizadas: Visual Studio, Estructuras de Datos
